Política de Privacidade

Pro Registro  ·  ProSoftware Informática Ltda.

Última atualização: 19 de junho de 2026

1. Sobre este documento

Esta Política de Privacidade descreve como o aplicativo Pro Registro, desenvolvido e operado pela ProSoftware Informática Ltda., coleta, utiliza, armazena e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com as práticas descritas neste documento.

2. Quem pode usar este aplicativo

O Pro Registro é um aplicativo corporativo (B2B) destinado exclusivamente a colaboradores e clientes de empresas do setor da construção civil que utilizam o sistema Pro Obra. O acesso requer credenciais fornecidas pela empresa contratante. Não é possível criar uma conta diretamente pelo aplicativo.

O aplicativo não é direcionado a menores de 18 anos.

3. Dados coletados e finalidade

O aplicativo coleta e utiliza os seguintes tipos de dados:

3.1 Dados de autenticação

  • Login e senha fornecidos pelo usuário para acesso à plataforma Pro Obra.
  • Token de sessão gerado após autenticação, armazenado de forma criptografada no dispositivo e removido ao encerrar a sessão.

3.2 Dados da empresa e obra

  • Informações da construtora selecionada (nome, URL da API) e da obra/unidade ativa, armazenadas localmente no dispositivo para manter a sessão entre usos.

3.3 Câmera e galeria de fotos

  • Câmera: utilizada para fotografar documentos, andamentos de obra e registros de chamados. As imagens são enviadas ao servidor da construtora vinculada.
  • Galeria (leitura): permite selecionar imagens existentes para anexá-las a chamados ou registros.
  • Galeria (gravação): permite salvar arquivos e fotos baixados do servidor da construtora no dispositivo do usuário.

3.4 Face ID / Biometria

  • Utilizado opcionalmente para autenticação rápida e segura. Nenhum dado biométrico é coletado ou transmitido pelo aplicativo; o processamento ocorre inteiramente pelo sistema operacional iOS.

3.5 Notificações

  • Notificações locais são utilizadas para informar o usuário sobre o andamento de downloads de arquivos. Nenhuma notificação é enviada remotamente sem ação do usuário.

4. Dados que NÃO coletamos

  • Localização geográfica
  • Contatos do dispositivo
  • Dados de saúde ou atividade física
  • Informações de outros aplicativos instalados
  • Identificadores de rastreamento publicitário
  • Dados de navegação ou comportamento para fins de marketing

O aplicativo não realiza rastreamento cross-app e não compartilha dados com redes de publicidade ou plataformas de análise de terceiros.

5. Compartilhamento de dados

Os dados inseridos ou gerados no aplicativo são transmitidos exclusivamente para os servidores da empresa contratante (construtora) através da API do sistema Pro Obra, operada pela ProSoftware Informática Ltda. Nenhum dado é vendido, cedido ou compartilhado com terceiros para fins comerciais.

6. Armazenamento e segurança

  • Toda comunicação entre o aplicativo e os servidores utiliza protocolo HTTPS.
  • O token de autenticação é armazenado com criptografia no dispositivo via Keychain (iOS).
  • Dados de sessão são removidos do dispositivo ao realizar logout.
  • Nenhum dado sensível é armazenado em texto simples.

7. Retenção de dados

Os dados armazenados localmente no dispositivo (token, informações de sessão) são mantidos apenas durante o período de uso ativo e excluídos ao encerrar a sessão. Dados armazenados nos servidores da empresa contratante estão sujeitos à política de retenção definida pela própria empresa.

8. Direitos do usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), os usuários têm direito a:

  • Confirmar a existência de tratamento de dados pessoais
  • Acessar os dados coletados
  • Solicitar correção de dados incompletos ou inexatos
  • Solicitar a exclusão dos dados tratados pelo aplicativo
  • Revogar consentimentos concedidos

Para exercer esses direitos, entre em contato pelos canais indicados na seção 10.

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. Quaisquer alterações relevantes serão comunicadas por meio de atualização da data no topo deste documento. Recomendamos a revisão periódica desta página.
